AssistMD Troubleshooting

Microphone

If you see “Microphone blocked”, click the mic permission icon in Chrome’s URL bar and allow access. You can also enable it under Chrome → Settings → Privacy and security → Site settings → Microphone.

Iframes and Popups

Mapping inside same-origin iframes is supported. For popup editors, map the field while the popup is focused. Multiple popups are handled via a chooser.

Insert Blocked

If you see “Confirm patient before inserting”, click Confirm patient in the panel. Switching charts mid-session writes a boundary marker and requires confirmation again.

Content Script Not Active

If mapping doesn’t respond, reload the EHR tab. The extension programmatically injects its content script when needed. Use Settings → Map Fields again after reload.

Recovery

Settings → Recovery shows the latest snapshot. Use Force Restore Snapshot to rehydrate mappings, templates, flags, and transcript. Cmd/Ctrl+B downloads a backup.
